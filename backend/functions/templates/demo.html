Our project, Gib, was inspired by the financial inclusivity that is core to the Web Monetization philosophy and its revolutionary architecture. Gib is designed to use the Web Monetization standard and the Interledger Protocol in a simple component - facilitating the sharing of premium web content. Think of Gib as a way to share pre-paid links to content.

To learn more about the Interledger Protocol, check out: https://interledger.org/developers/get-started/.
